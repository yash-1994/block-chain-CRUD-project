🚀 **Blockchain CRUD Project README** 🚀

## Prerequisites

1. Node.js
2. Ganache
3. Remix IDE

## Project Setup

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Configure all starting varible with your RPC URL and private key.

## Deploying the Smart Contract

1. Write your smart contract in Remix.
2. Deploy it to get the contract address.

## Running Ganache

1. Open Ganache.
2. Ensure it uses the same network as your RPC URL.

## Running the Project

1. Run: `index.html` using Live Server.
2. Perform CRUD operations.

That's it! Enjoy your blockchain project. 🌟
